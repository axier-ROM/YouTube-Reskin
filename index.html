How to install 02 / 03
01
Download the repo

Clone or download the extension files from GitHub. No account or API key required — just the folder.

git clone github.com/axier-ROM/YouTube-Reskin
02
Enable developer mode

Navigate to Chrome's extension page and flip the developer switch. Takes three seconds.

chrome://extensions → Developer mode: ON
03
Load the folder

Click "Load unpacked", point to the extension folder, and your YouTube immediately transforms. Done.

Load unpacked → Select folder → Done ✓
